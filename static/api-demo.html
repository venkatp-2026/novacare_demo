🏥 NovaCare Health API Demo

This demo shows how the frontend loads the API token from the environment via the backend.

How it works:
  1. Backend reads MOCK_API_TOKEN from environment variables
  2. Frontend calls /api/config to get the token
  3. Frontend uses the token for all subsequent API calls
